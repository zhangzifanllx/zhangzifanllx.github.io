如何将本地项目上传到gitee仓库首先新建一个仓库 填写完基本信息直接创建 将创建好的仓库克隆到本地步骤： 首先：复制仓库地址 在本地磁盘新建一个文件夹并打开文件夹，在空白处右击鼠标打开git bash here 这里我创建的文件夹是vueproject 输入命令 git clone https://gitee.com/sasukelike996/test.git 这里的第三个地址就是你刚才复制的仓库地址 仓库就克隆成功了（可以上传了） 这时候创建的文件夹下新增了一个文件夹，该文件夹的名称就是你仓库的名称 开始将项目进行上传在创建的文件夹下打开git Bash Here 使用命令git init 进行初始化 再打开你仓库的文件夹 依次执行命令 git add . git commit -m “这里写提交的信息” git push 最后在gitee中刷新查看仓库内容 即可完成上传的操作
